安装pygraphviz遇到的问题

环境：

MacOS sierra
python 2.7

使用安装命令：

sudo pip install pygraphviz

命令行报错，不能正确安装。报错日志如下：

看报错的消息应该是没有pkg-config这个命令，于是开始安装它。

brew install pkg-config

再次运行pygraphviz 安装命令，依然报错。

最后的大招

最后，经过各种google搜索和尝试，花费了我一上午的时间。才解决了这个问题。

此时graphviz成功安装。在python中就可以使用了

import pygraphviz as gv
